# Base Exam:

The following informations are documented in the Base Exam:

- **Eyelid abnormalities**

  - Blinking
  - Korb-Blackie Test
  - Hordeolum/Chalazion
  - Ektropium
  - Entropium
  - Lagophthalmus
  - Bells palsy
  - Floppy Eyelid
  - Lid laxity

- **Lashes**

  - Debris
  - Trichiasis

- **Skin related**

  - Rosazea
  - Lideczema

- **Others**

  - Canaliculitis
  - Dacryostenosis

- **Ant. Blepharitis**

  - Blepharitis
  - Demodex (Collarettes)
  - Teleangiectasia

- **Lid wiper**

  - Lid margin

- **Meibomian gland dysfunction**

  - Meibum quality
  - Expressibility of meibum
  - Meibomian gland orifices

- **Ocular Surface abnormalities**

  - Conjunctival redness
  - Conjunctival edema
  - LIPCOF/Conjunctivochalasis
  - Pterygium/Pincueculae
  - Fluorescein staining
  - Lissamine green staining
  - Corneal Sensitivity
  - Hornhautpathology
  - Filamentary keratitis

- **Intraocular findings**

  - Anterior chamber
  - Iris
  - Lens
  - Vitreous
  - Retina

- **Tests**

  - Schirmer test
  - InflammaDry MMP-9 Test
  - Osmolarity
  - Visual Acuity
  - IOP

- **Free text documentation**
