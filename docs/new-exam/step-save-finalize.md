# Step 3: Save / Finalize

Once the available information in each of the activable areas has been documented, you can "save" and/or "finalize" the base exam document by clicking each button in the menu at the bottom.

![SubStep Finalize Base Exam](/img/new-base-exam/newBasExa_save_finalize_buttons_en.png)

Once you have completed the **Base Exam** and confirmed that it is ready, click **Finalize**.

![SubStep Finalize Base Exam](/img/new-base-exam/newBasExa_save_finalize_popup_en.png)
