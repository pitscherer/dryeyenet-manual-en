# Patient Dashboard

Within the **Patient Dashboard**, you have access to all actions that can be performed for a patient:

- **Edit patient**
- **Invite to the Patient Portal**
- **View patient documentation**
- **Request treatment plan**

By clicking an action, it is either performed directly or a new screen is opened with the corresponding action selected.
