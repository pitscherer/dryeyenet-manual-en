# Step 5: Add files

Attach supporting files such as reports, PDFs, images, or device exports.

:::info Important operational notes

- Uploads are asynchronous.
- Wait for uploads to complete before moving forward.
- If you return to this step later, you can add more files from another folder.
- Maximum file size may be configured by your organization (often up to ~100 MB per file).
  :::
