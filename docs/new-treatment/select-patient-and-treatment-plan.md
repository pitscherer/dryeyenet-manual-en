# Click the **New Treatment** button in the Main Menu

In the main menu, click **New Treatment**.

![Main menu showing the New Treatment option](/img/new-treatment/newTre_maiMen_en.png)

# Select a Patient

After clicking **New Treatment**, a list of all patients will be displayed.

![List of patients](/img/new-treatment/newTre_selectPat_en.png)

# Select a Treatment Plan

If the patient has one or more treatment plans assigned, you will be able to select which treatment plan you want to document.

![List of treatment plans assigned to the patient](/img/new-treatment/newTre_selectTrePlan_en.png)
